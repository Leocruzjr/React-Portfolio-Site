#An interactive portfolio website.

Customize images by manipulating assets (listed in order that appears on site):
- hero banner img: assets/hero/heroImg.png
- img in about container: assets/about/aboutImg.png
- skill images: assets/skills/xxxx.png (use existing or upload your own) (this asset is called in data/skills.json. Make sure to change png in .json file)

Customize data:
- Experience section: data/history.json
- Projects section: data/projects.json
- Skill map/icons: data/skills.json
